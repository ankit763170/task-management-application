# Task Manager Application

A simple task management application built with React, allowing users to create, update, delete, and filter tasks. It provides a user-friendly interface to manage tasks efficiently.

## Features

- **Add New Tasks**: Create tasks with a title, due date, and status.
- **Update Existing Tasks**: Edit the title, due date, and status of tasks.
- **Delete Tasks**: Remove tasks from the list.
- **Search Tasks**: Find tasks by title using a search bar.
- **Filter Tasks**: Filter tasks by status (All, Pending, Completed).
- **Sort Tasks**: Sort tasks by due date or title for better organization.

## Technologies Used

- **Frontend**: 
  - React
  - Axios for API calls
  - CSS for styling
- **Backend**: 
  - Node.js
  - Express.js
  - Any preferred database (e.g., MongoDB, PostgreSQL)
- **Development Tools**: 
  - npm or yarn for package management
